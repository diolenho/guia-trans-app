# 🏳️‍⚧️ Guia Trans

Aplicação web desenvolvida para facilitar o acesso a informações, recursos e apoio para a população trans.

## 🚀 Acesse o projeto

👉 https://diolenho.github.io/guia-trans-app/#/

---

## 🧠 Sobre o projeto

O Guia Trans foi criado com o objetivo de centralizar informações importantes e melhorar a experiência de navegação de usuários em busca de recursos relacionados à comunidade trans.

O foco do produto está em:

- Acessibilidade  
- Clareza de navegação  
- Organização de informações  
- Experiência do usuário  

---

## 🎯 Problema

Usuários frequentemente enfrentam dificuldade para encontrar informações confiáveis e organizadas sobre serviços, apoio e conteúdos relevantes.

---

## 💡 Solução

Uma aplicação web simples e intuitiva que:

- Centraliza informações relevantes  
- Facilita a navegação por temas  
- Reduz fricções na jornada do usuário  
- Oferece uma experiência mais acessível  

---

## 🛠️ Tecnologias

- React  
- Vite  
- TypeScript  
- Tailwind CSS  
- React Router  

---

## 👨‍💻 Meu papel

- Definição do problema e proposta de valor  
- Estruturação da experiência do usuário  
- Desenvolvimento da aplicação (low-code + ajustes técnicos)  
- Deploy e resolução de problemas técnicos (build, routing, GitHub Pages)

## 🔧 Deploy

O projeto está hospedado via **GitHub Pages**, com build configurado via Vite e roteamento adaptado para funcionamento em ambiente estático.
